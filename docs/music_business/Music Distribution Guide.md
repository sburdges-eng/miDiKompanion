# Music Distribution Guide

Getting your music on streaming platforms and understanding how it all works.

---

## What Is Music Distribution

**Distribution = getting your music from you to listeners.**

**In the streaming era:**
- Distributors upload your music to platforms
- They handle the technical requirements
- They collect royalties from platforms
- They pay you

**You need a distributor to get on Spotify, Apple Music, etc.**

## Preparing Your Release

### Audio Requirements

| Requirement | Spec |
|-------------|------|
| Format | WAV preferred |
| Sample Rate | 44.1kHz (standard) or higher |
| Bit Depth | 16-bit minimum, 24-bit better |
| Levels | Peak below -1dB, -14 LUFS integrated typical |

### Metadata Essentials

**You need:**
- Song title (exact spelling matters)
- Artist name (consistent across releases)
- Album/single name
- Release date
- Genre (primary and secondary)
- Language
- Explicit content flag

### ISRC Codes

**International Standard Recording Code:**
- Unique ID for each recording
- Your distributor provides these
- Keep track of them
- Needed for royalty tracking

### UPC/EAN Codes

**For releases (albums, EPs, singles):**
- Your distributor provides these
- Different from ISRC
- One per release, not per song

## Artwork Requirements

### Technical Specs

| Requirement | Spec |
|-------------|------|
| Size | 3000x3000 pixels minimum |
| Format | JPG or PNG |
| Color | RGB |
| File size | Under 20-50MB typically |

### Content Rules

**DO:**
- Include artist/title (recommended)
- Make it readable at small sizes
- Be creative and memorable

**DON'T:**
- Use other people's images without rights
- Include explicit imagery (for non-explicit music)
- Use social media handles
- Include pricing or promotional text
- Use low-resolution images

## The Upload Process

### Step by Step

1. **Create release in distributor dashboard**
   - Enter release type (single, EP, album)
   - Set release date (2-4 weeks out minimum)

2. **Upload audio files**
   - WAV format preferred
   - Correct track order
   - Final masters only

3. **Enter metadata**
   - All credits
   - Song/album info
   - Genre tags

4. **Upload artwork**
   - High resolution
   - Correct dimensions

5. **Select platforms**
   - Usually select all
   - Consider timing for specific platforms

6. **Review and submit**
   - Double-check everything
   - Fix errors before submission

7. **Wait for review**
   - Usually 1-3 days
   - Can take longer for first release

## Understanding Royalties

### Types of Royalties

| Type | What It Is | Who Collects |
|------|------------|--------------|
| **Streaming royalties** | Payment per stream | Distributor |
| **Mechanical royalties** | Reproduction right | Publisher/PRO |
| **Performance royalties** | Public performance | PRO (ASCAP, BMI, etc.) |

### How Payment Works

```
Listener streams song
→ Platform pays distributor
→ Distributor takes cut (if any)
→ Distributor pays you
```

**Timing:** Usually monthly, 60-90 days after streams

### Maximizing Royalties

- Register with a PRO (performance royalties)
- Consider publishing admin (mechanical royalties)
- Track your metadata carefully
- Understand your splits with collaborators

## Common Problems

### Release Delayed

**Causes:**
- Metadata errors
- Artwork issues
- Audio problems
- First release (takes longer)

**Prevention:**
- Upload early
- Follow guidelines exactly
- Quality check everything

### Wrong Information Published

**Fix:**
- Update in distributor dashboard
- Can take days to propagate
- Some changes require takedown/re-upload

### Missing from Platform

**Check:**
- Is it live on other platforms?
- Did you select that platform?
- Contact support if issue persists

## Distribution Checklist

### Before Uploading

- [ ] Audio mastered and final
- [ ] Artwork ready (3000x3000 RGB JPG/PNG)
- [ ] Metadata written out
- [ ] Release date chosen (4+ weeks out)
- [ ] Credits confirmed with collaborators
- [ ] Explicit content flagged correctly

### After Uploading

- [ ] Review confirmation from distributor
- [ ] Claim artist profiles
- [ ] Set up pre-save campaign
- [ ] Plan promotional content
- [ ] Submit to Spotify editorial (4+ weeks before)

### After Release

- [ ] Verify live on all platforms
- [ ] Check metadata correct
- [ ] Monitor analytics
- [ ] Track royalties when they come in

## Platform-Specific Tips

### Spotify

- Submit to editorial 4+ weeks early
- Canvas videos increase engagement
- Artist playlists help algorithms
- Encourage saves (more valuable than streams)

### Apple Music

- Higher payouts
- Lossless/spatial audio options
- Less playlist-driven
- Good for album listeners

### YouTube Music

- Tied to YouTube video presence
- Upload music videos
- Optimise descriptions and tags
