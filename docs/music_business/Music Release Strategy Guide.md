# Music Release Strategy Guide

How to plan, execute, and promote your music releases effectively.

---

## The Modern Release Landscape

**What's changed:**
- Streaming dominates (Spotify, Apple Music)
- Singles > Albums for discovery
- Release often, stay visible
- Playlist placement matters
- Direct fan connection matters more

**What works now:**
- Consistent release schedule
- Strong single strategy
- Pre-release buzz building
- Multi-platform presence
- Authentic fan engagement

---

## Release Types

### Single

**Best for:**
- Building momentum
- Playlist pitching
- Testing new sounds
- Staying visible between projects

**Timing:**
- Release singles 4-6 weeks apart
- Build toward an EP or album
- Each single is a marketing moment

### EP (4-6 songs)

**Best for:**
- Establishing a cohesive sound
- Giving fans more depth
- Press coverage
- Selling/licensing opportunities

**Strategy:**
- Release lead single first
- EP drops 3-4 weeks later
- Second single after EP release

### Album (8-12+ songs)

**Best for:**
- Artistic statements
- Touring cycles
- Established artists
- Deep fan engagement

**Strategy:**
- Multiple singles before album
- Build anticipation over months
- Album release is the event

## Release Timeline

### 8-12 Weeks Before Release

**Week 12-8:**
- [ ] Music finalized (mixed, mastered)
- [ ] Artwork completed
- [ ] Metadata ready (credits, ISRC codes)
- [ ] Distributor selected
- [ ] Upload to distributor

### 4-8 Weeks Before Release

**Week 8-4:**
- [ ] Submit to Spotify for Artists for playlist consideration
- [ ] Create pre-save campaign
- [ ] Plan content calendar
- [ ] Reach out to press/blogs
- [ ] Brief your team (manager, publicist, label)
- [ ] Set up pre-order if applicable

### 2-4 Weeks Before Release

**Week 4-2:**
- [ ] Announce release date
- [ ] Begin social media teasers
- [ ] Send to playlist curators
- [ ] Pitch to influencers
- [ ] Finalize release day content
- [ ] Schedule posts

### 1 Week Before Release

**Week 1:**
- [ ] Final push for pre-saves
- [ ] Behind-the-scenes content
- [ ] Countdown posts
- [ ] Email newsletter
- [ ] Confirm all links work

### Release Day

- [ ] Post across all platforms
- [ ] Thank fans
- [ ] Engage with comments
- [ ] Share to stories
- [ ] Email blast
- [ ] Monitor for issues

### Post-Release

**Week 1-4 after:**
- [ ] Share fan reactions
- [ ] Release lyric video, acoustic version, remix
- [ ] Continued content (behind-the-scenes, stories)
- [ ] Analyze what worked
- [ ] Plan next release

## Distribution

### What Distributors Do

- Get your music on streaming platforms
- Collect royalties
- Provide analytics
- Handle metadata

### Distributor Options

| Distributor | Cost | Features |
|-------------|------|----------|
| DistroKid | ~$20/year | Unlimited releases, fast |
| TuneCore | Per release | Publishing admin available |
| CD Baby | Per release | Physical distribution |
| AWAL | Selective | Label services |
| Ditto | Annual fee | Good for UK/Europe |
| Amuse | Free tier | Basic free option |

### Choosing a Distributor

**Consider:**
- Cost structure (per release vs. annual)
- Royalty split (some take a cut, some don't)
- Additional services (sync, publishing)
- Customer support
- Payout timing

## Playlist Strategy

### Why Playlists Matter

**Playlist placement = discovery:**
- New listeners find you
- Streaming numbers increase
- Algorithm starts working for you
- Credibility boost

### Types of Playlists

| Type | Description | How to Get On |
|------|-------------|---------------|
| Editorial | Curated by platform staff | Pitch through distributor/platform |
| Algorithmic | Generated by algorithm | Get engagement, saves, listeners |
| User-created | Made by regular users | Reach out directly |
| Influencer | Created by tastemakers | Networking, submissions |

### Pitching to Spotify Editorial

**Through Spotify for Artists:**
1. Claim your artist profile
2. Go to Music tab
3. Submit unreleased music 4+ weeks before release
4. Write a compelling pitch

**Good pitch includes:**
- What the song is about
- What mood/genre it fits
- Recording story
- What makes it special
- Similar artists

### Building Playlist Relationships

- Find playlist curators on SubmitHub, PlaylistPush
- Engage genuinely (don't just spam links)
- Support other artists
- Build relationships before asking

## Pre-Save Campaigns

### What Pre-Saves Do

- Fans "save" before release
- Song appears in their library on release day
- Signals interest to algorithm
- Builds momentum

### Setting Up Pre-Saves

**Tools:**
- Feature.fm
- Hypeddit
- Toneden
- DistroKid Hyperfollow

**Best practices:**
- Offer incentive (exclusive content, merch discount)
- Make it easy (one click)
- Promote consistently
- Thank people who pre-save

## Content Around Releases

### Content Calendar

**Pre-release:**
- Announcement post
- Artwork reveal
- Preview clips (15-30 seconds)
- Behind-the-scenes clips
- Studio footage
- Lyric teasers
- Countdown posts

**Release day:**
- Release announcement
- Multiple posts across platforms
- Live stream or Q&A
- Stories throughout the day
- Fan engagement

**Post-release:**
- Thank you posts
- Fan reaction shares
- Alternative versions (acoustic, remix)
- Music video
- Lyric video
- Live performance video
- Continued content for weeks

### Video Content

| Type | Purpose | Timing |
|------|---------|--------|
| Teaser | Build anticipation | Pre-release |
| Lyric video | Accessible visual | Release day or week after |
| Music video | Main visual | With release or shortly after |
| Behind-the-scenes | Deepen connection | Throughout |
| Live performance | Show musicianship | Post-release |
| Vertical clips | Social media reach | Throughout |

## Press and Blogs

### Building Press List

**Find relevant:**
- Music blogs (Hype Machine, SubmitHub)
- Local press
- Genre-specific publications
- Online magazines
- Podcasts

### Press Kit Essentials

**Include:**
- Bio (short and long version)
- High-quality photos
- Private stream link
- Press quotes (if you have them)
- Social/streaming links
- Contact info

### Pitching Press

**DO:**
- Personalize each pitch
- Research the outlet
- Be concise
- Follow up once
- Accept rejection gracefully

**DON'T:**
- Mass email
- Be pushy
- Send huge attachments
- Pitch irrelevant outlets

## Release Day Checklist

**Morning:**
- [ ] Verify music is live on all platforms
- [ ] Check all links work
- [ ] Post to all social platforms
- [ ] Send email newsletter
- [ ] Thank pre-save supporters

**Throughout day:**
- [ ] Post to stories multiple times
- [ ] Engage with every comment
- [ ] Share fan posts/reactions
- [ ] Go live (optional)
- [ ] Send personal messages to key supporters

**Evening:**
- [ ] Reflection post
- [ ] Share streaming link again
- [ ] Thank your team
- [ ] Plan tomorrow's content

## Measuring Success

### Key Metrics

| Metric | What It Tells You |
|--------|-------------------|
| Streams | Reach and engagement |
| Saves | Fan investment |
| Playlist adds | Discovery potential |
| Follower growth | Converting listeners to fans |
| Shazams | Real-world discovery |
| Social engagement | Content effectiveness |

### Tools for Analytics

- **Spotify for Artists** — Detailed streaming data
- **Apple Music for Artists** — Apple insights
- **Chartmetric** — Cross-platform data
- **Soundcharts** — Industry analytics
- **Social platform insights** — Engagement data

### What Success Looks Like

**It depends on your stage:**
- First release: Any streams, some saves, learning
- Building: Growing month-over-month
- Established: Consistent numbers, playlist support

**Focus on growth rate, not absolute numbers.**

## Common Mistakes

| Mistake | Why It Hurts | Fix |
|---------|--------------|-----|
| Releasing without plan | No momentum | Plan 8+ weeks ahead |
| Not pitching playlists | Miss discovery | Submit 4+ weeks early |
| One post and done | Limited reach | Content for weeks |
| No pre-save | Miss day-one momentum | Set up campaign |
| Wrong timing | Competition | Check release calendar |
| No follow-up | Wasted momentum | Keep content going |
