INVOICE NAMING FORMAT
===============================================================================

   Lariat Invoice Jane Smith 11_20.xlsx

   Lariat Invoice Sarah Lee 10_17_25.xlsx  (with year)

‚ùå INCORRECT:
   Jane Smith Invoice.xlsx              (missing date)
   invoice_11_20.xlsx                   (missing client name)
   Lariat Invoice 11-20.xlsx            (wrong date format)

Format: [Optional: "Lariat Invoice" or "Invoice"] [ClientName] [M_D_YY].xlsx

===============================================================================
TYPICAL WORKFLOW
===============================================================================

AFTER EACH EVENT:
1. Save invoice to /INVOICE SPREADSHEETS/
2. Double-click lariat_automation.command
3. Choose Option 1 (Import New Invoices)
4. Database auto-updates!

END OF MONTH:
1. Double-click lariat_automation.command
2. Choose Option 2 (Generate Monthly Report)
3. Enter year and month
4. Report saved to /reports/ folder

===============================================================================
TROUBLESHOOTING
===============================================================================

"No new invoices found"
‚Üí Check files are in /INVOICE SPREADSHEETS/
‚Üí Verify file name format matches pattern
‚Üí Delete .processed_invoices.json to reprocess

"Error processing [filename]"
‚Üí Open file in Excel to verify it's not corrupted
‚Üí Check columns: A=Item, B=Cost, C=Amount, D=Total
‚Üí Verify totals rows exist (Sub total, Tax, Service Fee, Total)

"Permission denied"
‚Üí Run: chmod +x lariat_automation.command
