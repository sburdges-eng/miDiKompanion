## Types of MIDI Keyboards

### By Key Count

1. **Compact (25-37 keys)**
   - Portable, space-efficient
   - Ideal for producers on the go
   - Often include pads, knobs, faders

2. **Standard (49-76 keys)**
   - Balance between portability and range
   - Suitable for most production needs
   - Good for melodies and chords

3. **Full-Size (88 keys)**
   - Complete piano range
   - Often weighted or semi-weighted
   - Best for pianists and composers

### By Key Action

1. **Synth-Action**
   - Light, spring-loaded
   - Fast response
   - Ideal for electronic music

2. **Semi-Weighted**
   - Balance between synth and weighted
   - Moderate resistance
   - Versatile for various styles

3. **Weighted/Hammer-Action**
   - Mimics acoustic piano
   - Realistic feel
   - Best for pianists

### By Key Size

1. **Full-Size Keys**
   - Standard piano dimensions
   - Traditional feel
   - Better for experienced players

2. **Mini Keys**
   - Smaller, space-saving
   - More portable
   - May feel cramped for some

## Key Features & Technologies

### Core Features

- **Velocity Sensitivity** - Responds to playing dynamics
- **Aftertouch** - Pressure sensitivity after key press
- **Pitch Bend Wheel** - Continuous pitch control
- **Modulation Wheel** - Real-time modulation control
- **Transport Controls** - Play, stop, record, navigate DAW
- **Drum Pads** - Velocity-sensitive pads for beats/samples
- **Knobs/Faders** - Assignable controls for parameters
- **Display** - LCD/OLED for feedback and navigation

### Connectivity

- **USB-MIDI** - Direct computer connection (most common)
- **5-Pin DIN MIDI** - Traditional MIDI ports
- **Bluetooth** - Wireless control
- **CV/Gate Outputs** - For modular synthesizers
- **Pedal Inputs** - Sustain, expression pedals

### Software Integration

- **DAW Integration** - Pre-mapped controls for specific DAWs
- **NKS Support** - Native Instruments Komplete Kontrol
- **Bundled Software** - DAWs, plugins, virtual instruments
- **Template Support** - Pre-configured mappings

### Advanced Features

- **MPE (MIDI Polyphonic Expression)** - Individual note control
- **Arpeggiator** - Automatic pattern generation
- **Step Sequencer** - Pattern-based sequencing
- **Chord Mode** - One-finger chord playing
- **Scale Mode** - Constrains notes to scales
- **Smart Play** - Intelligent scale/chord assistance
