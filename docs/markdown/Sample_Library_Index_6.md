---

## Naming Convention

```
[BPM]_[Key]_[Type]_[Description]_[Number].wav

Examples:
120_Am_Kick_Punchy_01.wav
85_Cmaj_GuitarLoop_Acoustic_03.wav
_na_Riser_WhiteNoise_02.wav (no tempo/key)
```

---

## Tagging System

Use these tags to link samples to notes:

- `#sample/kick`
- `#sample/snare`
- `#sample/hihat`
- `#sample/bass`
- `#sample/synth`
- `#sample/guitar`
- `#sample/vocal`
- `#sample/fx`
- `#sample/loop`
