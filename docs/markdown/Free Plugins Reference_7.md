# Free Plugins Reference

Curated list of free AU plugins for Logic Pro on M4 Mac.

---

## Essential Must-Haves

| Plugin | Type | Notes |
|--------|------|-------|
| TDR Nova | Dynamic EQ | 4-band dynamic EQ, excellent for surgical work |
| Valhalla Supermassive | Reverb/Delay | Massive reverbs, delays, shimmer effects |
| Vital | Synth | Wavetable synth, rivals paid options |
| Analog Obsession LALA | Compressor | LA-2A emulation, great on vocals |
| MeldaProduction MFreeFXBundle | Multi | 37 free plugins, very comprehensive |

## Installation Notes

### AU Plugin Location
```
/Library/Audio/Plug-Ins/Components/
```

### After Installing
1. Restart Logic Pro
2. Open Preferences â†’ Plug-in Manager
3. Click "Reset & Rescan Selection" if needed
4. New plugins appear in channel strip inserts

### Apple Silicon Native
Most modern plugins support M4 natively. If not, Rosetta handles it.
