# Streamlit UI

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
cd DAiW-Music-Brain
streamlit run ui/emotion_to_logic.py
```

## Features

- **Simple Text Mode**: Quick generation from emotional text
- **Emotion Explorer**: Browse 216-node emotion thesaurus
- **Detailed Intent**: Full three-phase intent builder

## Modes

### Simple Text
Enter emotional description -> Analyze -> Generate -> Export

### Emotion Explorer
Browse hierarchy -> Select specific emotion -> Generate

### Detailed Intent
Build complete intent -> Generate with full control
