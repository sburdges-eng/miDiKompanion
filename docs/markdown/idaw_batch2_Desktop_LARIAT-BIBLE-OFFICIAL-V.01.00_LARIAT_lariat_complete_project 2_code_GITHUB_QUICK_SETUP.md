# Quick GitHub Setup for Lariat Bible

## 1. Create GitHub Repository (2 minutes)

1. Go to: https://github.com/new
2. Repository name: `lariat-bible`
3. Set to **PRIVATE**
4. DON'T initialize with anything
5. Click "Create repository"

## 2. Push Your Code (30 seconds)

Copy and paste these commands in Terminal:

```bash
cd ~/lariat-bible
git remote add origin https://github.com/YOUR_USERNAME/lariat-bible.git
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username.
