# LARIAT BANQUET INGREDIENT SYSTEM - COMPLETE OVERVIEW

## üìä SYSTEM COMPONENTS

### 1. **COMBINED GUIDE** (Main Worksheet)
- **Purpose**: Master product catalog with dual-vendor comparison
- **Structure**:
  - **SYSCO Products** (Columns D-K): Product #, Description, LIST#, Quantity, Amount, Size
  - **SHAMROCK Products** (Columns L-Q): Product #, Description, Quantity, Amount, Size
- **Usage**: Compare pricing and availability between two vendors
- **Key Feature**: Shows which items are exclusive to one vendor vs available from both

### 2. **SHEET2: INVENTORY/ORDERING**
- **Purpose**: Real-time inventory tracking and ordering system
- **Columns**:
  - Product Number
  - PAR Level (minimum stock level)
  - Current Inventory
  - Order Amount (auto-calculated based on PAR)
  - Description
  - Size
  - **LOCATION** (critical for kitchen operations)
- **Storage Locations Tracked**:
  - FREEZER (proteins, frozen items, churros, fries, etc.)
  - INSIDE WALK-IN (dairy, produce, fresh items)
  - KITCHEN (spices, dry goods, immediate-use items)
  - SHED (bulk items, disposables, dry storage)

### 3. **SHEET1: QUICK ORDER LIST**
- **Purpose**: Streamlined reordering
- **Format**: Simple 2-column list
  - Column A: Product Number
  - Column B: Quantity to Order
- **Usage**: Fast order generation without searching through full catalog

### 4. **RECIPE BOOK** (Separate Document)
- All recipes for your banquet menu items
- Includes: Sauces, Brines, Rubs, Dressings, Components
- Cross-references to ingredients in order guide

### WORKFLOW:

```
1. BANQUET EVENT BOOKED
   ‚Üì
2. SELECT MENU ITEMS (Passed Food/Buffet/Dessert)
   ‚Üì
3. REFERENCE RECIPE BOOK ‚Üí Determine ingredient needs
   ‚Üì
4. CHECK SHEET2 (INVENTORY) ‚Üí See what's in stock
   ‚Üì
5. COMPARE VENDORS (COMBINED GUIDE) ‚Üí Find best prices
   ‚Üì
6. CREATE ORDER (SHEET1) ‚Üí Quick order list
   ‚Üì
7. RECEIVE DELIVERY ‚Üí Update SHEET2 inventory levels
```

### KEY CONNECTIONS:

**Recipe Book ‚Üî Combined Guide**
- Match recipe ingredients to vendor product numbers
- Example: Recipe calls for "Heavy Cream" ‚Üí Find product #4218961 (Sysco) or #7182310 (Shamrock)

**Sheet2 ‚Üî Sheet1**
- Items below PAR level in Sheet2 should be added to Sheet1 order list
- Tracks quantity needed = (PAR level - Current inventory)

**Combined Guide ‚Üî Ordering**
- Compare Sysco vs Shamrock prices per item
- Select best vendor for each product
- Some items only available from one vendor (noted in guide)

**Storage Locations ‚Üî Kitchen Operations**
- Sheet2 LOCATION column tells kitchen staff exactly where to find ingredients
- Organized by temperature/storage requirements

## üí° BEST PRACTICES

### Inventory Management:
1. **Weekly Inventory Checks**: Update Sheet2 every week
2. **PAR Level Review**: Adjust PAR levels based on booking trends
3. **Location Verification**: Ensure items are stored in correct locations
4. **FIFO System**: First In, First Out - rotate stock properly

### Ordering Workflow:
1. **Monday**: Check Sheet2 for items below PAR
2. **Tuesday**: Compare vendors in Combined Guide
3. **Wednesday**: Finalize orders using Sheet1
4. **Thursday/Friday**: Receive deliveries, update inventory

### Recipe Integration:
1. **Event Confirmed**: Pull recipes from Recipe Book
2. **Scale Quantities**: Adjust for headcount
3. **Generate Ingredient List**: Cross-reference with Combined Guide
4. **Check Stock**: Verify inventory in Sheet2
5. **Order Shortfalls**: Add to Sheet1 order list
