### SONG OVERVIEW

**Tempo:** ~120 BPM (MIDI at 82 BPM)
**Key:** F major
**Guitar:** Arpeggiated sweep down/up | F voicing starts D string 3rd fret

**Chord Progressions:**
- **Verses/Loop:** F - C - Am - Dm *(the stuck, the cycle)*
- **Chorus/Weight:** F - C - Am - Bb *(the drop, the reveal)*
