## File Formats

| Format | Type | Use |
|--------|------|-----|
| WAV | Uncompressed | Recording, mixing |
| AIFF | Uncompressed | Mac equivalent of WAV |
| FLAC | Lossless compressed | Archiving, streaming |

| AAC | Lossy | Streaming, iTunes |

### Common Sample Rates
- 44.1 kHz — CD standard
- 48 kHz — Video/broadcast
- 96 kHz — High resolution
- 192 kHz — Studio high-res

### Common Bit Depths
- 16-bit — CD (96dB dynamic range)
- 24-bit — Professional (144dB range)
- 32-bit float — Internal processing
