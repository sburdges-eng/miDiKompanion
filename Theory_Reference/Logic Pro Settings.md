# Logic Pro Settings & Shortcuts

## Recommended Project Settings

### New Project Defaults
```
Sample Rate: 48 kHz (or 44.1 for CD-only projects)
Bit Depth: 24-bit
Frame Rate: (leave default unless for video)
```

### Audio Preferences
```
Output Device: PreSonus AudioBox iTwo
Input Device: PreSonus AudioBox iTwo
Buffer Size: 128 (recording) / 256-512 (mixing)
Process Buffer Range: Medium or Large
Multithreading: Playback & Live Tracks
```

## Essential Keyboard Shortcuts

### Transport
| Shortcut | Action |
|----------|--------|
| `Space` | Play/Stop |
| `Enter` | Go to beginning |
| `R` | Record |
| `C` | Cycle on/off |
| `,` | Rewind |
| `.` | Forward |
| `/` | Solo mode |

### Tracks
| Shortcut | Action |
|----------|--------|
| `M` | Mute track |
| `S` | Solo track |
| `A` | Show/hide automation |
| `Cmd+T` | New track |
| `D` | Create track with duplicate settings |
| `Cmd+D` | Duplicate region |

| Shortcut | Action |

## Custom Key Commands Worth Setting

These aren't defaults but are useful to set up:

| Function | Suggested Key |
|----------|---------------|
| Normalize Region Gain | `Ctrl+N` |
| Strip Silence | `Ctrl+X` |
| Create Marker | `'` (apostrophe) |
| Go to Marker | `Option+` number |

*Set in: Logic Pro → Settings → Key Commands*
