## Usage Example

```python
from cif_las_qef import UnifiedFramework, FrameworkConfig

# Create framework
config = FrameworkConfig(
    enable_cif=True,
    enable_las=True,
    enable_ethics=True,
    enable_qef=True
)
framework = UnifiedFramework(config)

# Create with emotional input
result = framework.create_with_consent(
    human_emotional_input={
        "biofeedback": {"heart_rate": 75, "eeg_alpha": 0.6},
        "voice": {"tone": 0.3, "intensity": 0.5},
        "intent": {"type": "creation"}
    },
    creative_goal={"style": "ambient"}
)
```
