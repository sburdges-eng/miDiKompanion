===============================================
LOGIC PRO PRODUCTION TEMPLATE - SETUP GUIDE
===============================================

TRACK 1: VOCALS (Input 1 - PreSonus AudioBox)
----------------------------------------------
Input: Input 1 (Mono)
Plugins (in order):
  1. Pitch Correction (Audio FX > Pitch > Pitch Correction)
     - Set Response to ~50ms for natural tuning
     - Set Scale to song key
  2. Channel EQ
     - High-pass at 80Hz
     - Cut 200-400Hz slightly (mud)
     - Boost 3-5kHz for presence
  3. Compressor
     - Ratio: 4:1
     - Attack: 10ms
     - Release: 100ms
  4. DeEsser
     - Target 5-8kHz range

TRACK 2: GUITAR (Input 2 - PreSonus AudioBox)
----------------------------------------------
Input: Input 2 (Mono)
Plugins (in order):
  1. Pedalboard (Audio FX > Amps and Pedals)
     - Add: Noise Gate, Tube Screamer, Chorus
  2. Amp Designer
     - Clean: Silverface
     - Crunch: Brit Gain
     - High Gain: Modern Stack
  3. Channel EQ
     - High-pass at 100Hz
     - Adjust mids to taste

TRACK 3: DRUMMER (Software Instrument)
----------------------------------------------
Type: Drummer Track
Kit: SoCal or East Bay (rock/pop)
Settings:
  - Complexity: 50%
  - Loudness: Match to song
  - Enable "Follow" on bass/rhythm track

TRACK 4: MIDI SYNTH (Optional)
----------------------------------------------
Type: Software Instrument
Input: MPK mini 3 (Channel 1)
Instrument: Alchemy or ES2

MIDI DRUM PROGRAMMING (Channel 10):
----------------------------------------------
Use MPK mini 3 pads for:
  Pad 1: Kick (C1)
  Pad 2: Snare (D1)
  Pad 3: Closed Hi-Hat (F#1)
  Pad 4: Open Hi-Hat (A#1)
  Pad 5-8: Toms/Cymbals

TO SAVE AS TEMPLATE:
----------------------------------------------
1. File > Save as Template...
2. Name: "Production Template"
3. Check "Include Audio"
4. Save

TO USE TEMPLATE:
----------------------------------------------
File > New from Template > My Templates > Production Template
